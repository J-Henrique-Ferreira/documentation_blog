Instalação do Sistema

Aprenda como instalar e configurar o sistema em seu ambiente de desenvolvimento ou produção.

Requisitos do Sistema

Antes de começar a instalação, certifique-se de que seu servidor atende aos seguintes requisitos:

  • PHP 8.1 ou superior
  • MySQL 5.7+ ou PostgreSQL 12+
  • Composer 2.0+
  • Node.js 16+ e NPM

Dica

Recomendamos usar o Docker para facilitar a configuração do ambiente de desenvolvimento.

Passo a Passo

1. Clone o repositório

git clone https://github.com/seu-usuario/seu-projeto.git
cd seu-projeto

2. Instale as dependências

composer install
npm install

3. Configure o ambiente

Copie o arquivo de exemplo e configure suas variáveis de ambiente: 

cp .env.example .env
php artisan key:generate

Atenção

Não se esqueça de configurar as credenciais do banco de dados no arquivo .env
